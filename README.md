# Saturn

Local-first operations runtime for leads, outreach, workflows, reporting, and monitoring.

## Read First
- Control contract: `.saturn/SATURN.md`
- Implementation spec: `docs/SATURN_CANONICAL_SPEC.md`

## Runtime Paths
- API: `backend/saturn-api.py`

## Core Rules
- Tool-first execution.
- SQLite first.
- Governed LLM queue only.
- Notion direct API only.
- No UI and no duplicate control layers.
