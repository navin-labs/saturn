# PURPOSE
Validate and restart the live Saturn service after safe local checks pass.

# WHEN TO USE IT
Use only after a patch that affects live runtime behavior or control contracts.

# WHAT IT MAY CHANGE
Service state, generated bytecode, and validated runtime processes.

# WHAT IT MUST NOT CHANGE
Secrets, schema, data records, or unrelated repo content.

# EXPECTED SAFE OUTPUT
Compile result, restart result, health result, and any blocked follow-up.

# FAILURE BEHAVIOR
Do not restart on failed validation; return the failing check and stop.
